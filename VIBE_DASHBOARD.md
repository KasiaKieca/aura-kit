# VIBE_DASHBOARD - AuraKit

## Product Vision
An interactive playground and starter kit for learning Vibe Coding, helping users translate natural language into functional apps using GitHub Copilot

## MVP Features (Must-have)
- [ ] "Vibe-first" structure with pre-configured Copilot instructions
- [ ] A library of proven prompts for planning and coding
- [ ] Example User Flow from initial idea to working prototype
- [ ] Checklist of best practices for maintaining flow state

## Roadmap
1. Documentation and prompt library preparation
2. React + Tailwind environment setup
3. Creating interactive "before and after prompt" examples
